# 🤖 Chatty AI - Desktop Chatbot

A simple desktop-based AI chatbot built using Python with a clean GUI interface. Chatty AI can respond to greetings, tell jokes, share fun facts, display the current time/date, provide weather information, and even generate playful flirt responses.

![Chatty AI Screenshot](assets/chatty-ai.png)

---

## ✨ Features

- 🕒 Current Time & Date
- 😂 Random Jokes
- 🎯 Fun Facts
- 💬 Greeting Responses
- ❤️ Flirt Mode
- 🌦️ Weather Information
- 🖥️ Simple Desktop GUI
- ⚡ Fast Response System

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/10ASB/AI-CHAT-BOT.git
cd AI-CHAT-BOT
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python chatbot.py
```

## 🎮 Available Commands

| Command | Description |
|----------|------------|
| Hello | Greeting |
| Time | Shows current time |
| Date | Shows current date |
| Fun fact | Random fact |
| Tell me a joke | Random joke |
| Flirt with me | Fun flirting response |
| Weather at loc | Weather information |
| Bye | Exit chatbot |
